# Page Type

Recognised page type keywords:

* `brands_page`
* `category_page`
* `contributor_page`
* `contributors_page`
* `content_item_article_page`
* `content_item_video_page`
* `content_item_static_page`
* `start_page`
* `tag_page`
* `tag_group_page`
